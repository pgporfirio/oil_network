## How to log

- Append one row per session at the end of the session.
- `Hours` is rounded to the nearest 0.5h. Estimate is fine — don't try to be precise.
- **Idle-gap rule:** if Pedro takes >5 minutes to reply to a question, treat the gap as not-working and subtract it from the session length. He's probably doing something else.
- `Notes` should be brief: which HANDOVER pass(es), or what was worked on if no pass shipped.
- Update the running total at the bottom whenever a row is added.
